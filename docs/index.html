extrudeFace

Extends a face along a direction vector by a given distance. Creates new geometry forming a prism-like shape. The original face is removed.

Signature

extrudeFace(struct, face, direction, distance, tolerance)

Parameters

struct: HalfedgeDS - The halfedge data structure

face: Face - The face to extrude

direction: Vector3 - Extrusion direction (normalized internally)

distance: Number - Extrusion distance

tolerance: Number - Vertex position comparison tolerance (default: 1e-10)

Returns

topFace: Face - The new extruded top face

sideFaces: Face[] - New side faces created

newVertices: Vertex[] - New vertices created during extrusion

Process

  1. Duplicate face vertices and offset along normalized direction vector
  2. Create new top face from duplicated vertices
  3. Create side faces connecting original face to new top face
  4. Remove original face from mesh

insetFace

Creates a smaller face inside the original by moving vertices inward toward the face centroid. Forms a recessed area. The original face is removed.

Signature

insetFace(struct, face, distance, tolerance)

Parameters

struct: HalfedgeDS - The halfedge data structure

face: Face - The face to inset

distance: Number - Inset distance (vertices move toward centroid)

tolerance: Number - Vertex position comparison tolerance (default: 1e-10)

Returns

insetFace: Face - The new inset face

sideFaces: Face[] - New side faces created

newVertices: Vertex[] - New vertices created during inset

Process

  1. Compute face centroid
  2. Move each vertex toward centroid by inset distance
  3. Create new inset face from new vertices
  4. Create side faces connecting original face to inset face
  5. Remove original face from mesh
